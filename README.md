## Descrição
Um sistema que armazena anotações dos usuários

## Tecnologias
- Next.js 14+ (App Router)
- MongoDB Atlas & Mongoose
- HeroUI (NextUI) & Tailwind CSS
- Next AUTH

## Funcionalidades
- [x] Cadastro e Login de Usuários (A autenticação não está funcional, apenas a criação dos usuários)
- [x] CRUD de Anotações


## Configuração
1. Clone o repositório
2. Configure o arquivo `.env` com a STRING DE CONEXÃO DO MONGO.
3. `npm install`
4. `npm run dev`
